# Web Application

A simple web application with Python server and HTML frontend.

## Structure

```
web_app/
├── server.py       # Python HTTP server
├── index.html      # Frontend HTML
└── README.md       # This file
```

## Usage

Start the server:

```bash
python3 server.py
```

Open your browser and navigate to:
http://localhost:8000

## Features

- Simple HTTP server
- Responsive web interface
- Static file serving
- CORS support
